# In Silico Analysis of TP53 Mutations in Colorectal Cancer

## 📌 Project Overview
This repository contains a Python-based computational data analysis project designed to visualize the relationship between *TP53* gene mutation status and patient age in colorectal cancer. 

## 🧬 Biological Context
The *TP53* gene is a critical tumor suppressor, and its mutation is a well-known driver in oncology. This project computationally investigates the clinical distribution of these mutations to see if patients with a mutated *TP53* gene present with cancer at a significantly different average age compared to those with the wildtype (normal) gene.

## 💻 Tools & Technologies Used
* **Language:** Python
* **Data Manipulation:** Pandas
* **Data Visualization:** Seaborn, Matplotlib
* **Environment:** Jupyter Notebook / Google Colab
* **Data Source:** Simulated clinical dataset modeled after TCGA (The Cancer Genome Atlas) clinical data.

## 📊 Key Findings
The generated visualization (Boxplot) demonstrates the age distribution across the two genetic cohorts. The computational analysis indicates that patients harboring a *TP53* mutation tend to show a lower median age of diagnosis compared to the wildtype cohort, highlighting the aggressive nature of the mutation.
